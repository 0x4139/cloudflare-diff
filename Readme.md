# cloudflare-diff
a small utility that updates the zone of a given domain based on the `dnsZone.json` file
## Installation

`npm i --save cloudflare-diff`
## Usage

`DOMAIN=0x4139.com EMAIL=0x4139@gmail.com TOKEN=your_token node cloudflare-diff/index.js`
